# Text Diff Viewer

A React-based web app (built with Vite) that compares two texts and highlights differences:

- **Red** → deleted words
- **Green** → added words

## Features

- Word-by-word text comparison
- Highlights deletions and additions clearly
- Clean and responsive UI

## Tech Stack

- React (via Vite)
- JavaScript
- CSS
